# ocptf

## build

```bash
$ make build-ocpsanity

```

## run

```bash
$ ./build/ocpsanity

```

use release version:

```
$ curl -LO https://github.com/hongkailiu/test-go/releases/download/0.0.8/ocpsanity-0.0.8-Linux-x86_64.tar.gz
$ tar xzvf ocpsanity-0.0.8-Linux-x86_64.tar.gz
$ ./ocpsanity/build/ocpsanity

```
